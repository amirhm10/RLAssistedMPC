\section*{Executive Summary}

\begin{itemize}
\item Distillation horizon adaptation is currently the strongest family in the
      report. The April 16, 2026 DQN and dueling runs both beat the paired MPC
      baseline, and the dueling variant is more concentrated around a small set
      of horizon recipes.
\item Distillation SAC weights are also positive on the newer reward profile,
      with final gap \(+1.520\) and last-50 gap \(+2.272\).
\item Distillation reidentification is still the most mixed result: strong
      mid-run collapse followed by recovery to a positive final gap \(+0.060\).
\item Polymer dueling horizon is still the best polymer disturbance result on
      disk. It beats the paired MPC baseline in 190 of 200 episodes and ends
      with final gap \(+1.692\).
\item Polymer weights are also clearly positive, with final gap \(+1.515\) and
      last-50 gap \(+1.732\).
\item Polymer DQN horizon remains positive and clearly beats MPC, but it is now
      weaker than the latest polymer dueling and weights runs.
\item Polymer reidentification is still slightly negative at the end, with final
      gap \(-0.070\), so polymer remains a weak benchmark for evaluating the
      model-update idea.
\end{itemize}

\section*{Van de Vusse Motivation And Current Stage}

The current motivation for Van de Vusse is still the same: polymer is giving a
good signal for horizon and weight adaptation, but it is still a weak benchmark
for the model-update idea. Polymer reidentification remains slightly worse than
MPC, and the removed matrix-style families were also not giving strong gains.
That makes it hard to judge whether the adaptation idea itself is weak or
whether the polymer benchmark is simply not sensitive enough.

Van de Vusse is therefore being developed as a benchmark where operating-point
changes should create a clearer need for model adaptation. The current stage is
still pre-RL:
\begin{itemize}
\item nonlinear plant added under \texttt{systems/vandevusse/}
\item direct-local-linearization system identification in place
\item corrected benchmark feed-temperature handling
\item offset-free MPC baseline notebook running in disturbance mode with the
      canonical \texttt{ca0\_blocks} disturbance
\item observer-update cleanup and observer-pole search utilities added
\item no Van de Vusse RL result bundle yet, so it is not part of the tables
\end{itemize}
